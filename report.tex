\section{Overview}
This lab implements a Fullstack Role-Based Access Control (RBAC) system using Node.js, Express, and React. The system features three distinct roles with specific permissions:
\begin{itemize}
    \item \textbf{SUPER\_ADMIN}: Full system control.
    \item \textbf{ADMIN}: Moderate user management (USER role only).
    \item \textbf{USER}: Restricted access to own profile only.
\end{itemize}
